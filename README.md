# Tampermonkey 实用脚本集合

> 这里收集了一些个人编写的油猴脚本，主要是一些写着玩玩的脚本，感觉有一点点实用（谦虚）。

## 简介

这些脚本主要用于增强网页功能，提升浏览体验。所有脚本均为个人学习和工作中的需求而编写，欢迎试用和反馈。

## 脚本列表

### 网站分类

| 文件夹 | 网站 | 描述 |
|--------|------|------|

## 快速开始

1. 安装 [Tampermonkey](https://tampermonkey.net/) 浏览器扩展
2. 选择您需要的脚本，点击查看具体安装说明
3. Enjoy!

## 注意事项

- 部分脚本可能与网站原有功能冲突
- 脚本更新可能滞后于网站版本变化
- 使用前请仔细阅读脚本说明
