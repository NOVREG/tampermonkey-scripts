# Bilibili  Scripts

> bilibili   Enhancements for Bilibili

##  Scripts

### 1. Double Click Fullscreen

**File:** `bilibili-double-click-fullscreen.user.js`

**Description:**
- Double-click the video player area to enter browser fullscreen mode
- Double-click again to exit fullscreen
- Supports both new and old Bilibili player versions

**Installation:**
1. Install [Tampermonkey](https://tampermonkey.net/) extension
2. Click the extension icon -> "Add a new script"
3. Copy the content of `bilibili-double-click-fullscreen.user.js`
4. Save the script
5. Visit any Bilibili video page, double-click the player area to fullscreen

**Features:**
- Uses native browser Fullscreen API
- Supports dynamically loaded players
- Complete error handling

## Compatibility

- Chrome
- Firefox
- Edge
- Safari
